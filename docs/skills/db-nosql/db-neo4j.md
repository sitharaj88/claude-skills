# Neo4j

Expert guidance for Neo4j graph modeling, Cypher queries, Graph Data Science algorithms, recommendation engines, and knowledge graphs.

## Usage

```bash
/db-neo4j [description or question]
```

## What It Does

1. Designs graph models with labeled nodes, typed relationships, and property schemas
2. Writes Cypher queries for pattern matching, path finding, and subgraph extraction
3. Applies GDS algorithms including PageRank, community detection, and node similarity
4. Builds recommendation engines using collaborative filtering on graph structures
5. Creates knowledge graphs with ontologies, entity resolution, and inference rules
6. Optimizes query performance with indexes, query profiling, and data modeling adjustments

## Examples

```bash
/db-neo4j model a social network with friend recommendations
```

```bash
/db-neo4j write a Cypher query to find shortest paths between entities
```

```bash
/db-neo4j build a knowledge graph for a product catalog with GDS analytics
```
