# MongoDB

Expert guidance for MongoDB data modeling, aggregation pipelines, indexing, Mongoose schema design, and Atlas configuration.

## Usage

```bash
/db-mongodb [description or question]
```

## What It Does

1. Designs document schemas with embedding vs. referencing trade-off analysis
2. Builds aggregation pipelines with stages like `$lookup`, `$unwind`, `$group`, and `$facet`
3. Recommends index strategies including compound, text, wildcard, and partial indexes
4. Creates Mongoose schemas with validation, middleware, virtuals, and plugins
5. Configures Atlas clusters, search indexes, and change streams
6. Implements sharding strategies and replica set configurations

## Examples

```bash
/db-mongodb model a social media app with posts, comments, and likes
```

```bash
/db-mongodb build an aggregation pipeline for sales analytics
```

```bash
/db-mongodb set up Atlas Search for a product catalog
```
