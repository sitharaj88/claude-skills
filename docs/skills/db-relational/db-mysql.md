# MySQL

Expert guidance for MySQL schema design, InnoDB optimization, stored procedures, query tuning, and replication setups.

## Usage

```bash
/db-mysql [description or question]
```

## What It Does

1. Designs schemas with appropriate storage engines, character sets, and collations
2. Writes efficient queries using joins, subqueries, and common table expressions
3. Creates stored procedures, functions, triggers, and events
4. Tunes InnoDB buffer pool, query cache, and connection settings for performance
5. Configures source-replica replication, Group Replication, and InnoDB Cluster
6. Manages partitioning strategies for large tables

## Examples

```bash
/db-mysql design an e-commerce schema with proper indexing
```

```bash
/db-mysql create a stored procedure for inventory management
```

```bash
/db-mysql configure source-replica replication for read scaling
```
