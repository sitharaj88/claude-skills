# Supabase

Expert guidance for Supabase PostgreSQL schemas, Row Level Security policies, Edge Functions, Realtime subscriptions, and Storage rules.

## Usage

```bash
/db-supabase [description or question]
```

## What It Does

1. Designs PostgreSQL schemas with tables, views, and functions via Supabase migrations
2. Writes Row Level Security policies for multi-tenant and role-based access control
3. Creates Edge Functions in Deno for serverless API endpoints and webhooks
4. Configures Realtime subscriptions for live updates on database changes
5. Sets up Storage buckets with access policies for file uploads
6. Integrates the Supabase client with Next.js, React, Flutter, and mobile apps

## Examples

```bash
/db-supabase design RLS policies for a multi-tenant project management app
```

```bash
/db-supabase create an Edge Function for Stripe webhook processing
```

```bash
/db-supabase set up Realtime subscriptions for a collaborative editor
```
