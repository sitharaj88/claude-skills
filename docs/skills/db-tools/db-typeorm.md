# TypeORM

Expert guidance for TypeORM decorator-based entities, QueryBuilder, repository patterns, migrations, and NestJS integration.

## Usage

```bash
/db-typeorm [description or question]
```

## What It Does

1. Defines entities using TypeScript decorators with columns, relations, and indexes
2. Writes queries using QueryBuilder for complex joins, subqueries, and raw SQL
3. Implements the repository and custom repository patterns for data access
4. Creates and runs migrations for schema changes across environments
5. Configures data sources for PostgreSQL, MySQL, SQLite, and other drivers
6. Integrates with NestJS modules, services, and dependency injection

## Examples

```bash
/db-typeorm create entities for an order management system with relations
```

```bash
/db-typeorm write a QueryBuilder query with pagination and sorting
```

```bash
/db-typeorm set up TypeORM with NestJS using custom repositories
```
