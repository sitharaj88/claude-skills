# Prisma

Expert guidance for Prisma schema design, type-safe queries, migrations, relation modeling, middleware, and database seeding.

## Usage

```bash
/db-prisma [description or question]
```

## What It Does

1. Designs Prisma schemas with models, relations, enums, and composite types
2. Writes type-safe queries using Prisma Client including nested reads and writes
3. Creates and manages migrations with `prisma migrate dev` and `prisma migrate deploy`
4. Configures middleware for logging, soft deletes, and query transformation
5. Builds seed scripts for development and testing environments
6. Integrates Prisma with Next.js, Express, and other frameworks

## Examples

```bash
/db-prisma design a schema for a blog with users, posts, and comments
```

```bash
/db-prisma write a nested create query with transaction support
```

```bash
/db-prisma set up seed script with realistic test data using faker
```
