# Firebase

Expert guidance for Firestore data modeling, security rules, Cloud Functions, real-time listeners, and composite index configuration.

## Usage

```bash
/db-firebase [description or question]
```

## What It Does

1. Models Firestore collections and documents with subcollection vs. root collection trade-offs
2. Writes security rules with granular read/write permissions and custom claims validation
3. Creates Cloud Functions for triggers, scheduled tasks, and callable endpoints
4. Implements real-time listeners with optimistic UI updates and offline persistence
5. Configures composite indexes for compound queries and collection group queries
6. Integrates Firebase with React, Next.js, Flutter, and Swift applications

## Examples

```bash
/db-firebase model a chat app with rooms, messages, and presence tracking
```

```bash
/db-firebase write security rules for a multi-role document approval workflow
```

```bash
/db-firebase create Cloud Functions for user onboarding and notification triggers
```
